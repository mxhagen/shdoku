
# 💡 shdoku

A basic tui sudoku game for your shell.

---

### Controls

The control scheme adheres to vim-like keybindings:

- `h, j, k, l` to move `left, down, up, right`
- `H, J, K, L` to move 3 spaces at once
- `x` to delete a number
- `1-9` to place a number
- `q` to quit
